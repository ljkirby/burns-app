# Assets Directory

This directory contains images and other static assets for the Burns App.

## Required Files

Please add the following files to this directory:

### burn-protocol.png
- The burn injury treatment protocol flowchart
- Should show the standard approach and ongoing assessment for burn injuries
- Used in the Protocol tab of the app (Burns Protocol toggle)

### cold-protocol.png
- The hypothermia/cold injury treatment protocol flowchart
- Should show the standard approach and ongoing assessment for cold injuries
- Used in the Protocol tab of the app (Hypothermia Protocol toggle)

### Optional App Icons (for production)
- `icon.png` (1024x1024) - App icon
- `splash.png` (2048x2048) - Splash screen
- `adaptive-icon.png` (1024x1024) - Android adaptive icon
- `favicon.png` (48x48) - Web favicon
